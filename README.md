# SehatQ Assessment

To install deps

```bash
yarn
```

To start dev server

```bash
yarn start
```

### Prerequisites

Dont forget to create a new `.env` file first, and include the following:

```bash
NODE_PATH=src
```

### Routes

-   `/` -- Home
-   `/search` -- Search page
-   `/history` -- Purchase history
-   `/product/:slug` -- Basically the product details page
-   `/login` -- Login page (soon)
